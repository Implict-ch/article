# 竞赛免机考 · 图文 PPT（3:4）

从「第一章 · 机考备考规划 · 第一节课」免试条件页的竞赛列表拆出，做成竖版 3:4（1080×1440）纯展示稿。

## 页序

| 页 | 内容 |
|----|------|
| 0 | 封面：华为秋招如何免机考？ |
| 1 | 软件算法 + 信息数理化 |
| 2 | 数学 + 网络安全 |
| 3 | 人工智能 + 超算计算 |
| 4 | 会议论文/期刊论文 |

封面对齐「目标院校讲解」样式；内容页顶部保留主标题「竞赛免机考」。

## 启动

```bash
cd presentation
npm install
npm run dev
```

默认地址：http://localhost:5182/

操作：点击 / 空格 / → 翻页；← 回退；底部圆点跳转。右侧可调条目字号。

## 导出截图

需先 `npm run dev`，再在上一级运行：

```bash
node _screenshot-slides.cjs
```

输出到 `截图/`（1080×1440）。
