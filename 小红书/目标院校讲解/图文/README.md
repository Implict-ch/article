# 华为目标院校 · 图文 PPT（3:4）

从「第一章 · 投递须知」目标院校清单页拆出，做成竖版 3:4（1080×1440）纯展示稿，适合小红书图文截图。

## 页序

| 页 | 内容 |
|----|------|
| 0 | 封面：华为校招目标院校名单 / 哪些学校更受青睐？（第五节课封面布局） |
| 1 | 京津东北 + 杭州厦门 |
| 2 | 山东江苏 + 粤港澳 |
| 3 | 上海合肥 + 西安 |
| 4 | 武汉长沙 + 成渝 |

封面为大标题；地区页顶部保留主标题「华为目标院校」。

右侧设置面板可拖动调节学校名称字号（会记住上次设置）。

## 启动

```bash
cd presentation
npm install
npm run dev
```

默认地址：http://localhost:5180/

操作：点击 / 空格 / → 翻页；← 回退；底部圆点跳转。

## 导出截图

需先 `npm run dev`，再在上一级运行：

```bash
node _screenshot-slides.cjs
```

输出到 `截图/`（1080×1440，学校字号 29px）。
