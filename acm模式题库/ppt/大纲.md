# ACM 模式题库宣传演示 HTML 设计稿

> 框架对齐 `文章/hot100笔记/ppt`：封面 → 起因 → 成果 → 题解结构 → 反馈 → CTA → 三连。  
> 叙事对齐已验证的 Hot100 破千赞结构：**作品发布体**（公共名词 + 形态）→ 为什么做 → 功能导览。

## 全局说明

- **用途**：录制讲解视频的演示骨架，配合录屏和后期剪辑使用
- **总页数**：8 页（含空白起幅 + 三连），全屏幻灯片式，键盘/点击翻页
- **视口**：16:9（1920×1080），固定尺寸不需要响应式
- **风格**：`paper-warm-gray`，对接现有 CSS 变量和字体系统
- **动画**：页面切入时元素可有轻微 fade-in / slide-up，不要花哨转场
